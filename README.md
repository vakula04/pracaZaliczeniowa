# Kalkulator C++

Prosty program konsolowy do obliczeń matematycznych.

## Funkcje
* Obsługa: `+`, `-`, `*`, `/`.
* Zabezpieczenie przed dzieleniem przez zero.
* Czysty kod w jednym pliku `main.cpp`.

## Instrukcja
1. Kompilacja: `g++ main.cpp -o kalkulator`
2. Uruchomienie: `./kalkulator`

## Git Workflow
* Rozwój na branchu: `feature/calculator`.
* Dokumentacja w `README.md`
